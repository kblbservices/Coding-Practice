Universal Selector


indicated by *
will apply to EVERY element on the page

Element Selector

- particular elements selected. can also style 2 or more elements
using a comma inbetween selectors ie. h1, h2 {}
ID Selector - can be used in an element attribute as a hook for css
in the CSS the id seletor is indicated with #id name

Class Selector


like the id selector, but used to style a group of elements
in the html, the class will be indicated as an attribute to the chosen group of elements called class=
in the CSS the class seletor is indicated with .class name

Descnedant Selectors


Used to select all elements that are nested in another element
Unlike styling 2 different elements separated by a comma, Descendatnts are
separated by a space ie li a {color: red} - the li will not be styled,
the anchor tags nested insilde a list will be styled
